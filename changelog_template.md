## 2026-03-14

### 决策：项目产品方向确定
- 项目定位为：网页端、面向高中生物教学场景的动态示意图编辑器
- 明确不是通用动画软件
- 明确第一阶段要优先服务老师快速制作课堂演示

- 支持：
  - 自由扩散
  - 协助扩散
  - 主动运输

### 决策：技术主路线确定
- 前端：
  - React
  - TypeScript
  - Vite
  - SVG
- 后端：
  - NestJS
  - TypeScript
  - Prisma
  - PostgreSQL

### 决策：Python 角色调整

- Python 保留给后期专项能力

### 决策：数据库选型确定

### 决策：动画组织方式确定
